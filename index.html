Проект "Оракул"

Машинная оценка рисков приказного производства в экономических судах

Сделано в dataШколе сообщества Открытые данные для Беларуси.
Источник данных: Интернет-портал судов общей юрисдикции Республики Беларусь.
Репозиторий проекта: github.com/ushchent/orakul.
Статус проекта: Испытательная версия.
Редактор проекта: Алексей Медвецкий, am@opendata.by.
